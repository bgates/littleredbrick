Six Steps to Setup

  1. Sign up by choosing how many teacher account subscriptions to purchase. You can try LittleRedBrick for free for 30 days. There is no risk. If you are a lone teacher, you can have a free account forever.
  2. Create a school by entering an account name for everyone at your school to use to log in, like selinsgrove.littleredbrick.com.
  3. Set dates for the term and marking periods to help automatically calculate grades.
  4. Name subjects so you can create class sections.
  5. Enter users including administrators, teachers, and students. You can create an unlimited number of administrator and student accounts (and parent accounts are created automatically for each student). Only teachers cost money. Type user names, copy them from a document, or upload them from a spreadsheet.
  6. Assign schedules for teachers. You can enter teacher schedules manually or by uploading the data from a spreadsheet, or have teachers enter their schedules themselves.
  7. Enroll students or have teachers do that themselves.
