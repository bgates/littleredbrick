Little Red Brick: Schools Online Simply

Analyzing Class Performance

LittleRedBrick takes the assignment and grade data teachers enter and assembles it into a comprehensive overview of each class. This compact, intuitive presentation allows teachers and administrators to understand a great deal about the academic performance of a class at a glance.

Grades

The class average and distribution for the marking period are calculated automatically based on the individual scores entered by the teacher and presented graphically. The grade scale for the class can be the default A-F depicted here or a different one set by the teacher.

Assignments

A quick summary of the kinds of work the teacher is assigning and how frequently the results are entered online, with a link to more information.

Discussions

A snapshot of the activity level in any discussion groups the teacher has set up for the class.

How You Can Use This

As an administrator, you can view these summaries for multiple related classes side-by-side.
