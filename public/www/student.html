Analyzing Student Performance

LittleRedBrick takes the assignment and grade data teachers enter and assembles it into a comprehensive overview of each student's academic performance. This informative summary provides teachers, administrators, students, and parents a complete picture of how a student is doing in a single class.

Notes

  • A link to the class performance page is provided for staff members only
  • The gradebook link is available only for the class teacher
  • The performance by category shows the relative importance of different kinds of assignments, and whether a student is struggling with a particular type
  • See changes in a student's grade over the course of the marking period with the grade progression chart
