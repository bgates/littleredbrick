Little Red Brick: Schools Online Simply

Calendar

LittleRedBrick provides an individualized calendar for everyone at your school who has a user account. Student calendars automatically include due dates for every assignment created by their teachers. In addition, different kinds of users can see and add different kinds of events to the calendar:

Administrators
can put events on the calendar for the whole school to see
Teachers
are able to enter events for their classes
Students
may see school events and the class events that were created by their teachers
Parents
may see all the events on their child's calendar
Everyone
can create personal events for the calendar which no other user will be able to see

Calendar Features

  • Assignment due dates show up on student/parent/teacher calendars when teacher creates assignment
  • All users can add personal events only they can see
  • Users can filter calendar to see only certain kinds of events (specific classes, personal-only, schoolwide events, etc)
